# test-umi-plugin

Test util for umi plugin.

## Install

Install as dev dependency.

```bash
$ yarn add test-umi-plugin -D
```

## Usage

```js
const testUmiPlugin = require('test-umi-plugin');
testUmiPlugin(options);
```

## Options

### fixtures

* Type: String

### basePort

* Type: Number
* Default: 12400
